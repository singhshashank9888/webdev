Lesson 1: Constants and Variables

In C programming, constants and variables are fundamental components of your code. They are used to store and manage data, and they play a crucial role in writing programs.

Constants

A constant is a value that cannot be changed during the execution of a program. It is typically used to store values that remain the same throughout the program's execution.

            
            const int max_attempts = 3;
            const double pi = 3.14159265359;

Variables

A variable, on the other hand, is a storage location that can hold different values during the program's execution. You can think of a variable as a named container for data.

            
            int score = 100;
            double temperature = 98.6;
            char grade = 'A';

Declaration and Initialization

In C, you must declare variables before using them. This involves specifying the data type and, optionally, providing an initial value.

            
            int count;            // Declaration without initialization
            double price = 29.99; // Declaration with initialization

Rules for Naming Constants and Variables

When naming your constants and variables, follow these rules:

  • Names must start with a letter or underscore (_).
  • Names can include letters, digits, and underscores.
  • Names are case-sensitive (e.g., 'score' and 'Score' are different).
  • Avoid using C keywords as names (e.g., 'int' or 'for').
